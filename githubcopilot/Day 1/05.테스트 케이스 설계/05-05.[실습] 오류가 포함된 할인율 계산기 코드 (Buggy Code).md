### 실습: 오류가 포함된 할인율 계산기 코드 (Buggy Code)

본 코드는 비즈니스 규칙을 바탕으로 작성되었으나, 의도적으로 논리적 오류와 경계값 처리 실수가 포함되어 있습니다. 수강생들은 이 코드를 바탕으로 GitHub Copilot과 함께 버그를 찾아내고 수정하는 실습을 진행합니다.

---

#### 1. 실습용 소스 코드

**파일명: discount_calculator.py**

def calculate_total_discount(amount, age, quantity):
    """
    비즈니스 규칙:
    1. 금액별 할인: 5만원 이상 5%, 10만원 이상 10%
    2. 연령별 추가 할인: 19세 미만 10%, 65세 이상 15% (중복 가능)
    3. 제한 사항: 최대 수량 100개, 금액은 음수 불가
    """
    
    # [버그 1] 음수 금액에 대한 예외 처리가 누락됨
    # [버그 2] 최대 수량 100개 제한 로직이 누락됨
    
    discount_rate = 0
    
    # [버그 3] 경계값 오류: 50,000원 '이상'인데 '>'를 사용함
    if amount > 50000 and amount < 100000:
        discount_rate = 0.05
    elif amount >= 100000:
        discount_rate = 0.10
        
    total_discount = amount * discount_rate
    
    # [버그 4] 연령 할인 로직 오류: 19세 '미만'인데 '<='를 사용함
    if age <= 19:
        total_discount += amount * 0.10
    # [버그 5] 시니어 할인 연산자 오류: 65세 '이상'인데 '>'를 사용함
    elif age > 65:
        total_discount += amount * 0.15
        
    return amount - total_discount

#### 2. 수강생 실습 가이드 (Copilot 활용법)

수강생들은 아래의 단계를 따라 Copilot과 협업하여 코드를 정상화합니다.

**단계 1: 코드 분석 및 취약점 파악**
* 활동: 코드를 전체 선택한 후 측면 채팅창(Copilot Chat)에 질문합니다.
* 프롬프트: "이 코드는 상단의 비즈니스 규칙 주석을 기반으로 작성되었어. 논리적인 오류나 경계값 처리 실수를 찾아서 리스트로 정리해줘."

**단계 2: 경계값 테스트를 통한 버그 확인**
* 활동: /tests 명령어를 사용하여 앞서 설계한 에러 시나리오 체크리스트의 값들을 입력합니다.
* 프롬프트: "/tests 49,999원, 50,000원, 19세, 101개 수량 등 주요 경계값에 대한 테스트 코드를 작성해서 실행해줘."
* 결과: 테스트 결과가 'Fail'로 나타나는 지점을 확인합니다.

**단계 3: 코드 수정 및 리팩토링 (/fix)**
* 활동: 오류가 발견된 함수 블록을 선택하고 수정을 요청합니다.
* 프롬프트: "/fix 발견된 경계값 오류를 수정하고, 수량 제한(100개)과 음수 금액 입력 시 ValueError를 발생시키는 가드 로직을 추가해줘."
