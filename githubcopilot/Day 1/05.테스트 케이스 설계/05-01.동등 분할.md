### 블랙박스 테스트 기법: 동등 분할 (Equivalence Partitioning, EP)

동등 분할은 프로그램의 입력 데이터를 유사한 결과를 기대할 수 있는 그룹으로 나누고, 각 그룹에서 대표값 하나를 선택하여 테스트하는 효율 중심의 블랙박스 테스트 기법입니다.

---

#### 1. 주요 개념 및 목적
* **개념:** 입력 값의 영역을 유효한 값과 유효하지 않은 값의 영역(그룹)으로 나누고, 각 영역 내의 값은 프로그램에 의해 동일하게 처리된다고 가정합니다.
* **목적:** 무수히 많은 입력 데이터를 모두 테스트하는 대신, 각 영역의 대표값만 테스트함으로써 테스트 횟수를 획기적으로 줄이면서도 결함 발견 확률을 유지합니다.

---

#### 2. 테스트 영역 구분 (Partitioning)
효과적인 동등 분할을 위해서는 다음 두 가지 영역을 모두 고려해야 합니다.

* **유효 영역 (Valid Partition):** 시스템이 정상적으로 수용하고 처리해야 하는 입력 값들의 집합입니다.
* **비유효 영역 (Invalid Partition):** 시스템이 거부하거나 에러 메시지를 출력해야 하는 부적절한 입력 값들의 집합입니다.



---

#### 3. 실습 사례: 성적 처리 시스템
입력 조건: "0점에서 100점 사이의 정수만 입력 가능"

| 구분 | 입력 데이터 범위 | 테스트 대표값 (예시) | 예상 결과 |
| :--- | :--- | :--- | :--- |
| **비유효 영역 1** | 점수 < 0 | -10 | 에러 메시지 출력 |
| **유효 영역** | 0 <= 점수 <= 100 | 50 | 정상 등록 |
| **비유효 영역 2** | 점수 > 100 | 120 | 에러 메시지 출력 |
