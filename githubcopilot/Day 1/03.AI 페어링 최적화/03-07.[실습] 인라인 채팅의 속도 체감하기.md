# 실습: 인라인 채팅의 속도 체감하기 (Inline Chat)

본 실습은 코드 에디터 내부에서 즉시 실행되는 인라인 채팅(Inline Chat)의 신속함과 직관성을 직접 체감하는 과정입니다. 복잡한 명령이나 대화 없이도 현재 작성 중인 코드의 형태를 순식간에 변환하는 능력을 학습합니다.

---

## 1. 준비: 리팩토링 대상 코드 작성

아래는 정수 리스트에서 5보다 큰 숫자들만 골라 10을 곱한 뒤 새로운 리스트에 저장하는 일반적인 for 루프 코드입니다. 이 코드를 `refactor_mission.py` 파일에 작성합니다.

**파일명: refactor_mission.py**
```python
def process_numbers(data):
    # 5보다 큰 숫자에 10을 곱하여 새로운 리스트 생성
    result = []
    for num in data:
        if num > 5:
            result.append(num * 10)
    return result

test_data = [2, 7, 4, 9, 5, 12]
print(process_numbers(test_data))
```

---

## 2. 과정: 인라인 채팅을 통한 한 줄 리팩토링

수강생은 마우스와 단축키만을 사용하여 코드를 현대적인 파이썬 스타일로 변환합니다.

**단계 1: 로직 선택 및 인라인 채팅 호출**
1. `process_numbers` 함수 내부의 `result = []` 부터 `for` 루프 전체(4~5줄)를 마우스로 드래그하여 선택합니다.
2. 단축키를 입력하여 인라인 채팅창을 활성화합니다.
   - **Windows/Linux:** `Ctrl + I`
   - **macOS:** `Cmd + I`

**단계 2: 리팩토링 명령 입력**
1. 나타난 입력창에 다음 명령어를 입력하고 Enter를 누릅니다.
   > "이 for 루프를 list comprehension으로 한 줄로 바꿔줘."

**단계 3: 차분(Diff) 확인 및 수락**
1. 에디터에 나타나는 빨간색(삭제 예정)과 초록색(추가 예정) 코드를 비교합니다.
2. 제안된 내용이 의도와 일치한다면 **Accept** (또는 단축키 `Ctrl/Cmd + Enter`)를 눌러 반영합니다.

---

## 3. 결과 확인 및 논의

인라인 채팅을 통해 변환된 코드는 다음과 같이 간결해져야 합니다.

```python
def process_numbers(data):
    # 최적화된 리스트 컴프리헨션
    return [num * 10 for num in data if num > 5]
```
