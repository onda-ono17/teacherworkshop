### 실습: 측면 채팅을 통한 심층 분석 (Sidebar Chat)

본 실습은 대화형 인터페이스인 측면 채팅(Sidebar Chat)을 활용하여 파일 전체의 로직을 조망하고, 코드가 품고 있는 잠재적인 위험 요소나 예외 상황을 다각도에서 분석하는 능력을 배양하는 과정입니다.

---

#### 1. 준비: 분석 대상 코드 작성

파일을 읽어 숫자로 변환한 뒤 평균을 계산하여 다른 파일에 저장하는 비즈니스 로직입니다. 일부러 예외 처리를 누락하여 AI의 분석 능력을 시험할 수 있도록 구성했습니다.

**파일명: data_processor.py**
```python
def process_data_files(input_path, output_path):
    # 입력 파일에서 데이터를 읽어와 평균을 계산하고 결과를 저장함
    with open(input_path, 'r') as f:
        lines = f.readlines()
        
    data = [int(line.strip()) for line in lines]
    average = sum(data) / len(data)
    
    with open(output_path, 'w') as f:
        f.write(f"Average: {average}")

# 실행 예시 (경로가 올바르지 않으면 에러 발생)
# process_data_files("input.txt", "output.txt")
```

---

#### 2. 과정: 측면 채팅을 활용한 다각도 분석

수강생은 대화창을 통해 인라인 채팅으로는 수행하기 어려운 '전체 요약'과 '잠재적 버그 탐색'을 수행합니다.

**단계 1: 측면 채팅창 활성화**
1. IDE 왼쪽 또는 오른쪽 패널에서 Copilot Chat 아이콘을 클릭하거나 단축키를 사용합니다.
   - **Windows/Linux:** `Ctrl + Alt + I` (또는 `Ctrl + Shift + I`)
   - **macOS:** `Cmd + Shift + I`

**단계 2: 전체 로직 요약 요청**
1. 채팅창에 다음과 같이 질문합니다.
   > "이 파일 전체의 로직을 한눈에 알기 쉽게 요약해줘."
2. **확인 포인트:** Copilot이 파일 읽기 -> 정수 변환 -> 평균 계산 -> 파일 쓰기라는 단계별 흐름을 정확히 파악하는지 확인합니다.

**단계 3: 잠재적 예외 사항(Bug) 추출**
1. 채팅창에 이어서 질문합니다.
   > "이 로직을 처리할 때 발생할 수 있는 잠재적인 예외 사항(Exceptions) 3가지를 알려줘."
2. **확인 포인트:** Copilot이 다음과 같은 항목들을 제시하는지 확인합니다.
    - **FileNotFoundError:** 입력 경로에 파일이 없을 경우.
    - **ValueError:** 파일 내용 중 숫자가 아닌 데이터가 섞여 있을 경우.
    - **ZeroDivisionError:** 파일이 비어 있어(0개) 합계를 나눌 수 없을 경우.

---

#### 3. 결과 확인 및 논의

인라인 채팅은 짧은 코드 수정에는 강력하지만, 측면 채팅은 다음과 같은 면에서 더 유리함을 확인합니다.

* **가독성 높은 설명:** 결과물이 코드 수정에 그치지 않고, 서술형 문장이나 불렛 포인트 형태로 제공되어 구조적 이해를 돕습니다.
* **연속적인 대화:** "방금 알려준 3가지 예외 상황을 처리하는 코드를 작성해줘"와 같이 이전 질문의 맥락을 유지하며 심화 작업을 이어갈 수 있습니다.
* **전역적 시야:** 파일의 특정 줄이 아닌, 함수 전체와 파일 간의 상호작용(파일 입출력 등)을 분석하는 데 적합합니다.
