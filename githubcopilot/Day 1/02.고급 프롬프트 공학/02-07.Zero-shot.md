# 핵심 기술: Zero-shot Prompting

Zero-shot Prompting은 인공지능 모델에게 구체적인 예시(Example)를 제공하지 않고, 오직 지시사항(Instruction)만으로 결과물을 도출해내는 기법입니다.

## 1. 정의 및 개념
* **정의:** 모델이 사전에 학습한 방대한 데이터를 바탕으로, 추가적인 예시 없이 사용자의 명령을 즉석에서 해석하여 코드를 생성하는 방식입니다.
* **특징:** 별도의 학습 데이터나 패턴을 제공할 필요가 없으므로 가장 빠르고 간편하게 Copilot을 활용할 수 있는 방법입니다.

## 2. 작동 원리
GitHub Copilot은 수십억 줄의 공개 코드를 이미 학습했습니다. 사용자가 "특정 함수를 만들어줘"라고 지시하면, 모델은 내부적으로 학습된 확률 통계 모델을 사용하여 해당 지시에 가장 적합한 코드 구조를 예측하여 제안합니다.

## 3. 장점과 한계
* **장점:** * 프롬프트 작성이 매우 빠르고 효율적입니다.
    * 표준적인 알고리즘(정렬, 탐색)이나 대중적인 라이브러리 활용 시 매우 높은 정확도를 보입니다.
* **한계:** * 프로젝트 고유의 복잡한 비즈니스 로직이나 사내 특수 컨벤션을 따르기 어렵습니다.
    * 요구사항이 복잡해질수록 사용자의 의도와 다른 코드가 생성될 가능성이 있습니다.
