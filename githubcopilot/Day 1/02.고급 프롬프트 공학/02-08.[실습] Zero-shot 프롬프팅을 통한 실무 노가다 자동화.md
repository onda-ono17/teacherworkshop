# 실습: Zero-shot 프롬프팅을 통한 실무 노가다 자동화

본 실습은 Copilot의 Zero-shot 생성 능력을 활용하여, 개발자가 평소 가장 귀찮아하고 시간이 오래 걸리는 "반복적인 단순 작업"을 단일 프롬프트로 1초 만에 해결하는 경험을 제공합니다.

---

## 1. 실습 개요
* **목표:** 예시 코드(Few-shot) 없이 자연어 요구사항만으로 복잡한 정규식, 더미 데이터, 구조 변환 코드를 정확히 생성해 냅니다.
* **준비물:** Visual Studio Code 또는 JetBrains IDE, GitHub Copilot.

---

## 2. 실습 과제 목록

### 과제 1: 목업(Mock) 더미 데이터 대량 생성
**상황:** 개발 테스트를 위해 사실적인 한국어 기반의 테스트 데이터가 필요합니다.
* **프롬프트 입력 (측면 채팅):**
  한국인 개발자 이름, 010으로 시작하는 무작위 휴대폰 번호, 생성일자(ISO8601), 부서명(개발, 기획, 디자인 중 택1) 속성을 가진 사용자 정보 JSON 데이터를 10개 만들어줘. 번호는 하이픈을 포함해줘.
* **핵심 체크포인트:** 
  - 자연스러운 한국어 이름과 규칙에 맞는 전화번호가 생성되었는가?
  - 리스트 형태의 유효한 JSON 배열 포맷인가?

### 과제 2: 외우기 힘든 복잡한 정규식(Regex) 생성
**상황:** 당장 비밀번호 유효성 검사 로직을 작성해야 하지만 정규식 문법을 검색하기 귀찮습니다.
* **프롬프트 입력 (인라인 채팅 Ctrl+I):**
  영문 대소문자, 숫자, 특수문자(!@#$%^&*)가 각각 1개 이상 포함되고 8자리에서 16자리 사이인 비밀번호를 검증하는 정규식 문자열과 유효성 검사 함수를 작성해줘.
* **핵심 체크포인트:**
  - 생성된 정규식이 요구사항의 제약을 모두 방어하는가?
  - 정규식이 무엇을 의미하는지 Copilot이 주석으로 잘 설명했는가?

### 과제 3: 시각화 다이어그램 자동 생성 (Mermaid)
**상황:** 복잡한 로그인 프로세스를 기획자에게 시각적으로 보여줘야 합니다.
* **프롬프트 입력 (측면 채팅):**
  클라이언트, API 게이트웨이, 인증 서버, 사용자 DB 사이의 'OAuth 기반 로그인 데이터 흐름'을 나타내는 마크다운 기반의 Mermaid.js 시퀀스 다이어그램 코드를 작성해줘.
* **핵심 체크포인트:**
  - 생성된 코드를 복사하여 VS Code 마크다운 뷰어(또는 Mermaid 확장프로그램)로 켰을 때 실제 다이어그램 형태(Flowchart)로 눈에 보이는가?
