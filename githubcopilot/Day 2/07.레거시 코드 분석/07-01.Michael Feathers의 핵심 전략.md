# Michael Feathers의 레거시 코드 개선 전략

레거시 코드 전문가 Michael Feathers가 제시하는 핵심 전략을 학습합니다. 기존 시스템의 안정성을 유지하면서 코드를 점진적으로 개선하는 구체적인 방법론을 다룹니다.

---

## 1. 레거시 코드의 정의: "테스트가 없는 코드"

Michael Feathers는 코드의 품질이나 작성 시점보다 '테스트의 존재 여부'에 집중합니다.

* **정의:** 테스트가 없는 코드는 레거시 코드입니다.
* **핵심 철학:** 테스트가 없는 코드는 수정을 시도할 때 의도치 않은 부작용(Side Effect)이 발생했는지 알 수 없습니다. 따라서 기능을 변경하거나 리팩토링하기 전, 현재의 동작을 보존하는 테스트를 구축하는 것이 최우선 과제입니다.

---

## 2. 접합부 (Seams)

코드를 직접 수정하지 않고도 프로그램의 동작을 변경하거나 관찰할 수 있는 지점을 찾는 기법입니다.

* **개념:** 소스 코드를 편집하지 않고도 그 위치의 행위를 바꿀 수 있는 지점입니다.
* **목적:** 의존성이 강하게 결합된 클래스나 함수를 테스트 코드에서 분리(Decoupling)하기 위해 활용합니다.
* **유형:**
    - 전처리 접합부 (Preprocessing Seams)
    - 링크 접합부 (Link Seams)
    - 객체 접합부 (Object Seams): 가장 흔히 사용되며, 다형성을 이용해 실제 객체를 Mock 객체로 교체하는 지점입니다.



---

## 3. 특성 테스트 (Characterization Tests)

코드를 고치기 전, 현재 시스템이 실제로 어떻게 동작하고 있는지 기록하기 위해 작성하는 테스트입니다.

* **개념:** 코드의 '옳고 그름'이나 '설계 의도'를 묻지 않습니다. 대신 시스템에 특정 입력을 주었을 때 나오는 '현재의 출력값'을 그대로 기록합니다.
* **목적:** 기능을 개선하거나 리팩토링한 후에도 기존의 복잡하고 기묘한 동작들이 그대로 유지되는지 확인하는 '안전망' 역할을 합니다.
* **절차:** 1. 대상 코드 호출을 위한 테스트 케이스를 작성합니다.
    2. 단언문(Assert)에 임의의 값을 넣고 실행하여 실제 출력값을 확인합니다.
    3. 실제 출력값을 기반으로 테스트 코드를 수정하여 통과시킵니다.



---

## 4. Sensing & Wrapping 기법

기존 코드를 최소한으로 건드리면서 새로운 기능을 추가하거나 내부 정보를 파악하는 구체적인 전술입니다.

### 4.1. 감지 (Sensing)
* **개념:** 코드 내부에 직접 접근할 수 없을 때, 해당 코드가 다른 객체에 미치는 영향을 통해 내부 상태를 파악하는 기법입니다.
* **방법:** Mock 객체나 가짜 객체(Fake Object)를 주입하여, 대상 코드가 해당 객체의 메서드를 어떻게 호출하는지 관찰함으로써 내부 로직의 수행 여부를 감지합니다.

### 4.2. 포장 (Wrapping)
* **개념:** 기존 코드를 수정하지 않고, 그 주변을 감싸서 새로운 기능을 추가하는 방식입니다.
* **기법:**
    - **포장 메서드 (Sprout Method):** 기존 메서드 내부에 코드를 추가하는 대신, 새로운 로직을 별도의 메서드로 만들고 이를 호출하는 방식입니다.
    - **포장 클래스 (Wrapper Class):** 데코레이터 패턴과 유사하게, 기존 클래스를 멤버 변수로 가지는 새로운 클래스를 만들어 전후 처리를 추가하는 방식입니다.

---

## 5. 핵심 포인트 및 Copilot 활용

* **안전 우선:** "리팩토링의 첫 번째 단계는 코드를 깨끗하게 만드는 것이 아니라, 테스트를 만드는 것"임.
* **Copilot 활용:** - 레거시 코드를 Copilot에게 보여주며 "이 코드에서 의존성을 분리할 수 있는 접합부(Seam)를 찾아줘"라고 요청할 수 있습니다.
    - "이 복잡한 레거시 함수의 특성 테스트(Characterization Test)를 위한 다양한 입력 데이터 조합을 만들어줘"라고 지시하여 빠르게 안전망을 구축할 수 있습니다.
