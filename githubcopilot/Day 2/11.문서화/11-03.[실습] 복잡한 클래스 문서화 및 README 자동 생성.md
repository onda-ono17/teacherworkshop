## 1. 실습용 소스 코드 (Python: OrderManager)

수강생들은 의도를 파악하기 어려운 아래 코드를 분석하고 문서화해야 합니다.

**파일명: order_manager.py**

```python
class OrderManager:
    def __init__(self, db_connection):
        self.db = db_connection

    def p_process(self, u_id, i_list, d_code=None):
        # 주석이 전혀 없는 복잡한 주문 및 할인 로직
        r = {"status": "fail", "total": 0}
        u = self.db.get_user(u_id)
        if not u: return r
        
        t_amt = 0
        for item in i_list:
            p = self.db.get_price(item['id'])
            t_amt += p * item['qty']
            
        if d_code == "WINTER2026":
            t_amt *= 0.8
        elif u.is_vip:
            t_amt *= 0.9
            
        res = self.db.save_order(u_id, t_amt)
        if res:
            r["status"] = "success"
            r["total"] = t_amt
        return r
```
